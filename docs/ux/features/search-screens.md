## 4. 공통 컴포넌트

### 4.1 SearchBar
검색창 컴포넌트

```dart
SearchBar(
  query: String?,
  hintText: String,
  onQueryChanged: Function(String),
  onSubmitted: Function(String),
  onClear: VoidCallback,
  autofocus: bool,
)
```

### 4.2 RecentSearchItem
최근 검색어 아이템

```dart
RecentSearchItem(
  query: String,
  onTap: VoidCallback,      // 검색 실행
  onDelete: VoidCallback,   // 삭제
)
```

### 4.3 SearchSectionHeader
검색 결과 섹션 헤더

```dart
SearchSectionHeader(
  title: String,      // "노래", "아티스트" 등
  count: int,         // 검색 결과 건수
  onMoreTap: VoidCallback?,  // "더보기" 탭
)
```

### 4.4 SongSearchTile
노래 검색 결과 타일

```dart
SongSearchTile(
  song: Song,
  onTap: VoidCallback,
)
```

### 4.5 ArtistSearchTile
아티스트 검색 결과 타일

```dart
ArtistSearchTile(
  artist: Artist,
  isFollowing: bool,
  onTap: VoidCallback,
  onFollowTap: VoidCallback,
)
```

### 4.6 PostSearchTile
게시글 검색 결과 타일

```dart
PostSearchTile(
  post: Post,
  onTap: VoidCallback,
)
```

### 4.7 UserSearchTile
사용자 검색 결과 타일

```dart
UserSearchTile(
  user: User,
  isFollowing: bool,
  isCurrentUser: bool,
  onTap: VoidCallback,
  onFollowTap: VoidCallback?,
)
```

### 4.8 SearchEmptyView
검색 결과 없음 뷰

```dart
SearchEmptyView(
  query: String,
  category: SearchCategory?,  // null이면 전체
  suggestedKeywords: List<String>,
  onKeywordTap: Function(String),
)
```

## 5. AC 검증 체크리스트

### AC-F8-1: 통합 검색
- [ ] 검색창에 키워드 입력 가능
- [ ] 2글자 이상 입력 시 자동 검색
- [ ] 노래/아티스트/게시글/사용자 4개 카테고리 결과 표시
- [ ] 각 카테고리별 상위 N개 미리보기

### AC-F8-2: 카테고리별 필터
- [ ] 카테고리 탭 표시 (전체/노래/아티스트/게시글/사용자)
- [ ] 탭 선택 시 해당 카테고리만 표시
- [ ] "더보기" 탭 시 전체 목록 표시
- [ ] 탭 전환 시 결과 갱신

### AC-F8-3: 최근 검색어
- [ ] 검색 화면 진입 시 최근 검색어 표시
- [ ] 최대 10개 저장
- [ ] 최근 검색어 클릭 시 해당 키워드로 검색
- [ ] 개별 삭제 (X 버튼)
- [ ] 전체 삭제 기능

### AC-F8-4: 검색 결과 없음
- [ ] 결과 없을 시 Empty 상태 표시
- [ ] 적절한 메시지 표시
- [ ] 추천 검색어 제안 (선택)

### AC-F8-5: 검색 결과 상세 이동
- [ ] 노래 클릭 → 노래 상세 화면 (F2)
- [ ] 아티스트 클릭 → 아티스트 상세 화면 (F3)
- [ ] 게시글 클릭 → 게시글 상세 화면 (F5)
- [ ] 사용자 클릭 → 사용자 프로필 화면 (F7)

## 6. 데이터 요구사항 (참고용)

> 이 섹션은 Step 3(DB 설계)를 위한 참고 정보입니다.

### 6.1 검색 대상
- **노래**: 제목(한글/일본어), 아티스트명
- **아티스트**: 이름(한글/영문/일본어)
- **게시글**: 내용 텍스트
- **사용자**: 닉네임, 아이디(username)

### 6.2 최근 검색어
- 사용자별 최대 10개 저장
- 검색어, 검색 일시
- 로컬 저장 (SharedPreferences) 또는 서버 저장

### 6.3 검색 결과 응답
- 각 카테고리별 결과 리스트
- 총 건수
- 페이지네이션 정보 (더보기용)
