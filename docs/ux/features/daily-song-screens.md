# Daily Song (F2) - Screen Specifications

## 1. 화면 목록

| 화면 ID | 화면명 | Route | AC 커버리지 |

- **Route**: `/home`
- **진입점**: 앱 시작, 하단 탭 "홈"

### 2.3 UI 요소 상세

#### 2.3.1 AppBar
| 요소 | 타입 | 동작 |
|------|------|------|
| 로고/타이틀 | Text | "hibi" 또는 로고 이미지 |
| 프로필 아이콘 | IconButton | 프로필 화면으로 이동 |

#### 2.3.2 날짜 표시 영역
| 요소 | 타입 | 설명 |
|------|------|------|
| 날짜 (한국어) | Text | "2026년 2월 3일" 형식 |
| 요일 (일본어) | Text | "月曜日 (월요일)" 형식 |

#### 2.3.3 노래 카드
| 요소 | 타입 | 동작 | AC |
|------|------|------|-----|
| 앨범 이미지 | Image | 카드 전체 탭 시 상세 이동 | AC-F2-1 |
| 곡 제목 (한글) | Text | 표시 전용 | AC-F2-1 |
| 곡 제목 (일본어) | Text | 표시 전용 | AC-F2-1 |
| 아티스트명 | TextButton | 탭 시 아티스트 상세(F3) 이동 | AC-F2-1 |
| 앨범명 | Text | "앨범명 · 발매년도" 형식 | AC-F2-1 |
| 좋아요 버튼 | IconButton | 좋아요 토글 | AC-F2-2 |

#### 2.3.4 하단 탭바
| 탭 | 아이콘 | Route | 기능 |
|----|--------|-------|------|
| 홈 | 🏠 | `/home` | 오늘의 노래 |
| 캘린더 | 📅 | `/calendar` | 히스토리 (F4) |
| 아티스트 | 👤 | `/artists` | 아티스트 목록 (F3) |
| 설정 | ⚙️ | `/settings` | 설정 |

## 4. 공통 컴포넌트

### 4.1 SongCard
홈 화면에서 사용하는 노래 카드 컴포넌트

```dart
SongCard(
  song: DailySong,
  onTap: () => navigateToDetail,
  onLikeTap: () => toggleLike,
)
```

### 4.2 LikeButton
좋아요 버튼 컴포넌트 (애니메이션 포함)

```dart
LikeButton(
  isLiked: bool,
  onTap: VoidCallback,
  size: double,
)
```

### 4.3 ArtistChip
아티스트 정보를 표시하는 탭 가능한 컴포넌트

```dart
ArtistChip(
  artist: Artist,
  onTap: () => navigateToArtist,
)
```

## 5. AC 검증 체크리스트

### AC-F2-1: 오늘의 노래 표시
- [ ] 홈 화면(DS-01)에 오늘 날짜가 표시됨
- [ ] 노래 카드에 곡명(한글/일본어)이 표시됨
- [ ] 노래 카드에 아티스트명이 표시됨
- [ ] 노래 카드에 앨범 이미지가 표시됨

### AC-F2-2: 노래 좋아요
- [ ] 홈 화면(DS-01)에서 좋아요 버튼 클릭 가능
- [ ] 상세 화면(DS-02)에서 좋아요 버튼 클릭 가능
- [ ] 좋아요 상태가 즉시 UI에 반영됨
- [ ] 좋아요 상태가 화면 간 동기화됨

### AC-F2-3: 노래 상세 정보
- [ ] 상세 화면(DS-02)에 가사가 표시됨
- [ ] 상세 화면에 발매일이 표시됨
- [ ] 상세 화면에 앨범 정보가 표시됨
- [ ] 상세 화면에 아티스트 정보가 표시됨

### AC-F2-4: 추천곡 없음
- [ ] 홈 화면(DS-01)에 Empty 상태 UI가 표시됨
- [ ] "아직 추천곡이 없습니다" 메시지가 표시됨

## 6. 데이터 요구사항 (참고용)

> ⚠️ 이 섹션은 Step 3(DB 설계)를 위한 참고 정보입니다.

### 6.1 노래 정보
- 곡 ID
- 곡명 (한글)
- 곡명 (일본어/원제)
- 아티스트 정보 (ID, 이름, 이미지)
- 앨범 정보 (이름, 이미지, 발매일)
- 가사 (일본어, 한글 번역)
- 장르
- 추천 날짜
- 외부 링크 (Spotify, Apple Music, YouTube)

### 6.2 좋아요 정보
- 사용자 ID
- 노래 ID
- 좋아요 여부
- 좋아요 일시
