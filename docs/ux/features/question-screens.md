## QU-01: 문의 작성 화면

### 기본 정보
- **경로**: `/question/new`
- **접근 조건**: 로그인 필수
- **주요 AC**: AC-F10-1, AC-F10-2, AC-F10-3, AC-F10-6

## 데이터 모델

### Question (문의)
```dart
class Question {
  final int id;
  final int memberId;           // 작성자 ID
  final QuestionType type;      // 문의 유형
  final String title;           // 제목 (최대 100자)
  final String content;         // 내용 (최대 1000자)
  final QuestionStatus status;  // 상태
  final String? answer;         // 답변 (nullable)
  final DateTime? answeredAt;   // 답변 일시
  final DateTime createdAt;
  final DateTime updatedAt;
}

enum QuestionType {
  account,    // 계정
  service,    // 서비스 이용
  bug,        // 버그 신고
  feature,    // 기능 제안
  other,      // 기타
}

enum QuestionStatus {
  received,   // 접수됨
  processing, // 처리중
  answered,   // 답변완료
}
```

## 접근성

- 스크린 리더: 상태 뱃지에 "상태: 답변완료" 등의 레이블
- 키보드 네비게이션: Tab 키로 폼 필드 간 이동
- 충분한 터치 영역: 버튼 최소 높이 48dp
- 에러 메시지: 색상 외에도 아이콘으로 표시
