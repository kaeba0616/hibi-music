# F12: Management (관리자 기능) 화면 정의

## 화면 목록

| 화면 ID | 화면명 | 설명 |
|---------|--------|------|
| MG-01 | 관리자 대시보드 | 통계 요약 및 메뉴 |
| MG-02 | 신고 목록 | 신고 목록 조회 |
| MG-03 | 신고 상세 | 신고 상세 및 처리 |
| MG-04 | 문의 목록 | 문의 목록 조회 |
| MG-05 | 문의 상세 | 문의 상세 및 답변 |

| MG-07 | FAQ 편집 | FAQ 추가/수정 |

| MG-09 | 사용자 상세 | 사용자 상세 및 제재 |

## MG-01: 관리자 대시보드

### 기본 정보
- **경로**: `/admin`
- **접근 조건**: ADMIN 권한 필수
- **주요 AC**: AC-F12-1, AC-F12-11

│  │  대기중     │  │  미답변     │      │

│                                         │

### UI 요소

| 요소 | 타입 | 설명 |
|------|------|------|
| 신고 통계 카드 | Card | 대기중 신고 수, 탭 시 MG-02 이동 |
| 문의 통계 카드 | Card | 미답변 문의 수, 탭 시 MG-04 이동 |
| 회원 통계 카드 | Card | 총 회원 수, 탭 시 MG-08 이동 |
| FAQ 통계 카드 | Card | 총 FAQ 수, 탭 시 MG-06 이동 |
| 메뉴 목록 | ListTile | 각 관리 메뉴 진입점 |

## 접근성

- ADMIN 권한이 아닌 사용자가 관리자 URL 접근 시 메인 화면으로 리다이렉트
- 각 조치 전 확인 다이얼로그 표시
- 중요 작업(삭제, 정지, 탈퇴)은 되돌릴 수 없음을 명확히 안내

## 데이터 모델 (참고용)

### AdminStats (대시보드 통계)
```dart
class AdminStats {
  final int pendingReports;      // 대기중 신고 수
  final int unansweredQuestions; // 미답변 문의 수
  final int totalMembers;        // 총 회원 수
  final int totalFaqs;           // 총 FAQ 수
}
```

### MemberDetail (사용자 상세)
```dart
class MemberDetail {
  final int id;
  final String email;
  final String nickname;
  final String? profileImage;
  final String role;           // USER, ADMIN
  final String status;         // ACTIVE, SUSPENDED, BANNED
  final DateTime createdAt;
  final int postCount;
  final int commentCount;
  final int followerCount;
  final int followingCount;
  final int reportReceivedCount;
  final int reportSentCount;
}
```
