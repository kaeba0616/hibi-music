# Calendar (F4) - UX Flow

## 1. 개요

### 1.1 기능 설명
과거에 추천받은 JPOP 노래들을 월별 캘린더 형식으로 확인하고, 날짜별로 조회하는 기능

### 1.2 관련 AC
- AC-F4-1: 월별 캘린더 표시
- AC-F4-2: 날짜 선택
- AC-F4-3: 월 변경
- AC-F4-4: 좋아요 필터

---

## 2. 사용자 여정 (User Journey)

### 2.1 Entry Points
| 진입점 | 조건 | 목적지 |
|--------|------|--------|
| 하단 탭 "캘린더" 클릭 | 로그인 상태 | 캘린더 화면 (현재 월) |
| 홈 화면에서 "이전 추천곡 보기" | 로그인 상태 | 캘린더 화면 |

### 2.2 Exit Points
| 종료점 | 조건 | 다음 목적지 |
|--------|------|------------|
| 노래 카드 클릭 | 추천곡 선택 시 | 노래 상세 화면 (F2) |
| 아티스트명 클릭 | 추천곡 카드에서 | 아티스트 상세 화면 (F3) |
| 다른 탭 선택 | 어디서나 | 해당 탭 화면 |

## 4. 화면 목록

| 화면 ID | 화면명 | Route | 설명 |
|---------|--------|-------|------|
| CA-01 | 캘린더 화면 | `/calendar` (탭) | 월별 캘린더 + 선택된 날짜의 노래 표시 |

## 5. 주요 인터랙션

### 5.1 날짜 선택
```
[날짜 셀 탭]
    │
    ├─ 선택된 날짜 하이라이트 표시
    │
    ├─ 해당 날짜의 추천곡 API 호출 (또는 캐시 조회)
    │
    └─ 하단에 노래 카드 표시 (또는 Empty 메시지)
```

### 5.2 월 변경
```
[이전/다음 월 버튼 탭]
    │
    ├─ 월 표시 업데이트 (예: "2026년 2월")
    │
    ├─ 해당 월의 추천곡 목록 API 호출
    │
    ├─ 캘린더 마커 업데이트
    │
    └─ 선택된 날짜 초기화 (또는 유지)
```

### 5.3 좋아요 필터
```
[좋아요만 필터 토글]
    │
    ├─ 필터 상태 변경
    │
    ├─ 마커 업데이트 (좋아요한 날짜만 / 전체)
    │
    └─ 선택된 노래 카드 업데이트
```

### 5.4 오늘 버튼
```
[오늘 버튼 탭]
    │
    ├─ 현재 월로 이동
    │
    ├─ 오늘 날짜 선택
    │
    └─ 오늘의 추천곡 표시
```

## 6. 에러 처리

| 에러 상황 | 사용자에게 보여줄 내용 | 액션 |
|-----------|----------------------|------|
| 네트워크 에러 | "연결을 확인해주세요" | 재시도 버튼 |
| 서버 에러 | "잠시 후 다시 시도해주세요" | 재시도 버튼 |
| 좋아요 실패 | 토스트 "좋아요 처리 실패" | 자동 롤백 |

---

## 7. 특수 케이스

### 7.1 추천곡이 없는 날짜
- 마커가 표시되지 않음
- 날짜 선택 시 Empty 메시지: "해당 날짜에는 추천곡이 없습니다"

### 7.2 미래 날짜
- 선택 불가 (비활성화 상태)
- 회색 처리로 시각적 구분

### 7.3 좋아요 필터 적용 시 Empty
- 필터를 적용했지만 좋아요한 노래가 없는 경우
- 메시지: "좋아요한 노래가 없습니다"
- 서브 메시지: "노래에 좋아요를 눌러보세요!"
