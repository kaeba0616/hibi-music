## 인터랙션 상세

### FAQ 항목 토글
1. 접힌 상태: 질문만 표시, 우측에 `>` 아이콘
2. 펼친 상태: 질문 + 답변 표시, 우측에 `∨` 아이콘
3. 동시에 여러 FAQ 펼치기 가능 (독립적 동작)

### 검색 동작
1. 검색창에 포커스 → 키보드 표시
2. 키워드 입력 (300ms debounce)
3. 질문/답변 텍스트에서 키워드 검색
4. 매칭되는 FAQ만 필터링되어 표시
5. 검색어 초기화: `✕` 버튼 또는 검색창 비우기

### 카테고리 필터
1. 기본: "전체" 선택 상태
2. 카테고리 탭 클릭 → 해당 카테고리 FAQ만 표시
3. 검색어와 카테고리 필터 동시 적용 가능

## 데이터 모델

### FAQ
```dart
class FAQ {
  final int id;
  final String question;      // 질문
  final String answer;        // 답변
  final FAQCategory category; // 카테고리
  final int order;            // 정렬 순서
  final bool isPublished;     // 공개 여부
  final DateTime createdAt;
  final DateTime updatedAt;
}

enum FAQCategory {
  account,    // 계정
  service,    // 서비스 이용
  community,  // 커뮤니티
  other,      // 기타
}
```

## 접근성

- 스크린 리더: FAQ 항목에 "질문: {질문내용}, 탭하여 답변 보기" 레이블
- 키보드 네비게이션: Tab 키로 FAQ 항목 간 이동, Enter로 토글
- 충분한 터치 영역: FAQ 항목 최소 높이 48dp
