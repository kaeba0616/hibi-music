# Post (F5) - Screen Specifications

## 1. 화면 목록

| 화면 ID | 화면명 | Route | AC 커버리지 |
|---------|--------|-------|-------------|
| PO-01 | 게시글 목록 (피드) | `/feed` (탭) | AC-F5-1, AC-F5-7 |
| PO-02 | 게시글 작성 | `/posts/create` | AC-F5-2, AC-F5-4 |
| PO-03 | 게시글 상세 | `/posts/:postId` | AC-F5-3, AC-F5-6, AC-F5-7 |

## 3. PO-02: 게시글 작성 화면

### 3.1 기본 정보
- **Route**: `/posts/create`
- **진입점**: 피드 FAB, 하단 탭 길게 누름 등

### 3.3 UI 요소 상세

#### 3.3.1 AppBar
| 요소 | 타입 | 동작 |
|------|------|------|
| 닫기 버튼 (✕) | IconButton | 작성 취소 (확인 다이얼로그) |
| 타이틀 | Text | "새 게시글" |
| 등록 버튼 | TextButton | 게시글 등록 (AC-F5-2) |

#### 3.3.2 텍스트 입력
| 요소 | 타입 | 동작 |
|------|------|------|
| 텍스트 필드 | TextField | 본문 입력 (필수) |
| Placeholder | Text | "무슨 생각을 하고 계신가요?" |
| 글자 수 | Text | 현재/최대 (예: 0/500) |

#### 3.3.3 이미지 첨부 영역
| 요소 | 타입 | 동작 |
|------|------|------|
| 이미지 썸네일 | Image | 첨부된 이미지 표시 |
| 삭제 버튼 (✕) | IconButton | 이미지 제거 |
| 추가 버튼 (+) | IconButton | 이미지 추가 (최대 4장) |

#### 3.3.4 노래 태그 (AC-F5-4)
| 요소 | 타입 | 동작 |
|------|------|------|
| 노래 정보 | Card | 제목, 아티스트 표시 |
| 삭제 버튼 (✕) | IconButton | 태그 제거 |

#### 3.3.5 하단 툴바
| 요소 | 타입 | 동작 |
|------|------|------|
| 이미지 버튼 | IconButton | 이미지 선택 바텀시트 |
| 노래 태그 버튼 | IconButton | 노래 선택 바텀시트 (PO-05) |

## 5. PO-04: 게시글 수정 화면

### 5.1 기본 정보
- **Route**: `/posts/:postId/edit`
- **진입점**: 게시글 상세 더보기 메뉴 → 수정
- **인증 필요**: Yes (본인만)

### 5.2 화면 레이아웃
PO-02 (게시글 작성)와 동일한 레이아웃, 다음 차이점:
- AppBar 타이틀: "게시글 수정"
- 등록 버튼 → "저장" 버튼
- 기존 내용이 미리 채워져 있음

### 5.3 UI 요소 상세
PO-02와 동일, 추가 사항:
- 기존 텍스트 로드
- 기존 이미지 표시
- 기존 노래 태그 표시

## 8. AC 검증 체크리스트

### AC-F5-1: 게시글 목록 조회
- [ ] 피드 화면(PO-01)에 게시글 목록이 표시됨
- [ ] 각 카드에 작성자 정보가 표시됨
- [ ] 각 카드에 작성 시간이 상대 시간으로 표시됨
- [ ] 각 카드에 좋아요/댓글 수가 표시됨
- [ ] 무한 스크롤이 동작함

### AC-F5-2: 게시글 작성
- [ ] 작성 화면(PO-02)에서 텍스트 입력 가능
- [ ] 등록 버튼 클릭 시 게시글 생성
- [ ] 생성 후 목록에 새 게시글 표시

### AC-F5-3: 게시글 상세 조회
- [ ] 상세 화면(PO-03)에 전체 내용 표시
- [ ] 첨부 이미지가 표시됨
- [ ] 댓글 목록이 표시됨

### AC-F5-4: 노래 태그
- [ ] 노래 선택 바텀시트(PO-05)가 표시됨
- [ ] 노래 검색이 동작함
- [ ] 선택한 노래가 게시글에 첨부됨

### AC-F5-5: 게시글 수정
- [ ] 수정 화면(PO-04)에 기존 내용이 로드됨
- [ ] 저장 버튼 클릭 시 수정 완료

### AC-F5-6: 게시글 삭제
- [ ] 상세 화면(PO-03)에서 삭제 가능
- [ ] 삭제 확인 다이얼로그 표시
- [ ] 삭제 후 목록으로 이동

### AC-F5-7: 게시글 좋아요
- [ ] 목록(PO-01)에서 좋아요 토글 가능
- [ ] 상세(PO-03)에서 좋아요 토글 가능
- [ ] 좋아요 상태가 즉시 UI에 반영됨
