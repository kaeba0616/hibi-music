# Comment & Reaction (F6) - Screen Specifications

## 1. 화면 목록

F6는 별도 화면이 아닌 기존 게시글 상세 화면(PO-03)의 컴포넌트로 구현됩니다.

| 컴포넌트 ID | 컴포넌트명 | 위치 | AC 커버리지 |
|-------------|-----------|------|-------------|
| CO-01 | 댓글 섹션 | PO-03 하단 | AC-F6-1, AC-F6-5 |
| CO-02 | 댓글 입력창 | PO-03 하단 고정 | AC-F6-2, AC-F6-4 |
| CO-03 | 댓글 카드 | CO-01 내부 | AC-F6-1, AC-F6-3, AC-F6-5 |
| CO-04 | 대댓글 카드 | CO-03 하위 | AC-F6-4 |
| CO-05 | 댓글 삭제 다이얼로그 | 모달 | AC-F6-3 |

## 3. CO-01: 댓글 섹션

### 3.1 기본 정보
- **위치**: 게시글 상세 화면(PO-03) 하단
- **구성**: 섹션 헤더 + 댓글 목록

### 3.2 섹션 헤더

| 요소 | 타입 | 동작 |
|------|------|------|
| 댓글 아이콘 | Icon | 💬 |
| 댓글 수 | Text | "댓글 N개" |

### 3.3 상태별 UI

## 4. CO-02: 댓글 입력창

### 4.1 기본 정보
- **위치**: 화면 하단 고정 (SafeArea 포함)
- **구성**: 입력 필드 + 전송 버튼

### 4.2 UI 요소

| 요소 | 타입 | 동작 |
|------|------|------|
| 입력 필드 | TextField | 댓글 내용 입력 |
| Placeholder | Text | "댓글을 입력하세요..." |
| 전송 버튼 | IconButton/TextButton | 댓글 전송 |

### 4.3 상태별 UI

#### 4.3.1 기본 상태 (빈 입력)
```
┌─────────────────────────────────────┐
│  ┌─────────────────────────┐ [전송]│
│  │ 댓글을 입력하세요...     │       │  ← 전송 버튼 비활성화 (회색)
│  └─────────────────────────┘       │
└─────────────────────────────────────┘
```

#### 4.3.2 입력 중
```
┌─────────────────────────────────────┐
│  ┌─────────────────────────┐ [전송]│
│  │ 정말 좋은 곡이에요!      │       │  ← 전송 버튼 활성화 (Primary)
│  └─────────────────────────┘       │
└─────────────────────────────────────┘
```

#### 4.3.3 대댓글 모드
```
┌─────────────────────────────────────┐
│  @닉네임1님에게 답글                [✕] │  ← 대댓글 모드 표시 + 취소 버튼
├─────────────────────────────────────┤
│  ┌─────────────────────────┐ [전송]│
│  │ 저도 좋아해요!           │       │
│  └─────────────────────────┘       │
└─────────────────────────────────────┘
```

#### 4.3.4 전송 중
```
┌─────────────────────────────────────┐
│  ┌─────────────────────────┐  [◯] │  ← 로딩 인디케이터
│  │ 정말 좋은 곡이에요!      │       │  ← 입력 필드 비활성화
│  └─────────────────────────┘       │
└─────────────────────────────────────┘
```

## 5. CO-03: 댓글 카드

### 5.1 기본 정보
- **위치**: 댓글 섹션 내부
- **구성**: 프로필 + 내용 + 액션

### 5.2 레이아웃

```
┌─────────────────────────────────────────────┐
│  ┌────┐  닉네임              · 1시간 전     │
│  │사진│  정말 좋은 곡이죠! 저도 요즘 매일   │
│  └────┘  듣고 있어요 ❤️                     │
│          ♡ 3   [답글]                       │
└─────────────────────────────────────────────┘
```

### 5.3 UI 요소

| 요소 | 타입 | 동작 |
|------|------|------|
| 프로필 이미지 | CircleAvatar | 탭 시 사용자 프로필 이동 |
| 닉네임 | Text | 탭 시 사용자 프로필 이동 |
| 작성 시간 | Text | 상대 시간 표시 |
| 내용 | Text | 댓글 본문 |
| 좋아요 버튼 | IconButton | 좋아요 토글 (AC-F6-5) |
| 좋아요 수 | Text | 숫자 표시 |
| 답글 버튼 | TextButton | 대댓글 입력 모드 전환 (AC-F6-4) |
| 더보기 메뉴 | IconButton | 본인 댓글만 표시 (삭제) |

### 5.4 본인 댓글 vs 타인 댓글

#### 본인 댓글
```
┌─────────────────────────────────────────────┐
│  ┌────┐  나 (You)            · 2시간 전 [⋮]│  ← 더보기 메뉴 표시
│  │사진│  앨범도 추천해요~                   │
│  └────┘  ♥ 2   [답글]                       │  ← 본인도 좋아요 가능
└─────────────────────────────────────────────┘
```

#### 타인 댓글
```
┌─────────────────────────────────────────────┐
│  ┌────┐  닉네임              · 1시간 전     │  ← 더보기 메뉴 없음
│  │사진│  정말 좋은 곡이죠!                  │
│  └────┘  ♡ 3   [답글]                       │
└─────────────────────────────────────────────┘
```

### 5.5 삭제된 댓글 (대댓글이 있는 경우)

```
┌─────────────────────────────────────────────┐
│  삭제된 댓글입니다                           │  ← 회색 배경, 이탤릭
└─────────────────────────────────────────────┘
   ┌──────────────────────────────────────┐
   │ ┌────┐ 닉네임2          · 30분 전    │  ← 대댓글은 유지
   │ │사진│ @삭제됨 그래도 동의해요!       │
   │ └────┘ ♡ 1                           │
   └──────────────────────────────────────┘
```

## 6. CO-04: 대댓글 카드

### 6.1 기본 정보
- **위치**: 원 댓글 바로 아래
- **구성**: CO-03과 동일하나 들여쓰기 + 답글 버튼 없음

### 6.2 레이아웃

```
   ┌──────────────────────────────────────┐
   │ ┌────┐ 닉네임2          · 30분 전    │
   │ │사진│ @닉네임1 저도 동감이에요!      │
   │ └────┘ ♡ 1                           │  ← 답글 버튼 없음
   └──────────────────────────────────────┘
```

### 6.3 특징
- 왼쪽 16px 들여쓰기
- @원작성자 멘션 표시 (탭 시 해당 댓글로 스크롤)
- 답글 버튼 없음 (대대댓글 미지원)

## 8. 공통 컴포넌트

### 8.1 CommentCard
댓글/대댓글 카드 컴포넌트

```dart
CommentCard(
  comment: Comment,
  isReply: bool,           // 대댓글 여부
  onLikeTap: VoidCallback,
  onReplyTap: VoidCallback?,  // 대댓글은 null
  onDeleteTap: VoidCallback?, // 본인 댓글만
  onAuthorTap: VoidCallback,
  onMentionTap: VoidCallback?, // 대댓글만
)
```

### 8.2 CommentInput
댓글 입력창 컴포넌트

```dart
CommentInput(
  controller: TextEditingController,
  replyTo: Comment?,       // 대댓글 모드 시 원 댓글
  onSubmit: (String content) => void,
  onCancelReply: VoidCallback?,
  isLoading: bool,
)
```

### 8.3 CommentSection
댓글 섹션 전체 컴포넌트

```dart
CommentSection(
  postId: int,
  comments: List<Comment>,
  onCommentSubmit: (String content, int? parentId) => void,
  onCommentDelete: (int commentId) => void,
  onCommentLike: (int commentId) => void,
)
```

## 9. AC 검증 체크리스트

### AC-F6-1: 댓글 목록 조회
- [ ] 게시글 상세 화면에 댓글 섹션 표시
- [ ] 댓글마다 작성자 정보 표시 (프로필, 닉네임)
- [ ] 댓글마다 내용 및 작성 시간 표시
- [ ] 대댓글이 원 댓글 아래 들여쓰기되어 표시

### AC-F6-2: 댓글 작성
- [ ] 입력창에 텍스트 입력 가능
- [ ] 전송 버튼 클릭 시 댓글 생성
- [ ] 생성된 댓글이 목록에 즉시 표시
- [ ] 빈 입력 시 전송 버튼 비활성화

### AC-F6-3: 댓글 삭제
- [ ] 본인 댓글에만 더보기 메뉴 표시
- [ ] 삭제 확인 다이얼로그 표시
- [ ] 삭제 후 목록에서 제거
- [ ] 대댓글이 있는 원 댓글 삭제 시 "삭제된 댓글입니다" 표시

### AC-F6-4: 대댓글 작성
- [ ] 답글 버튼 탭 시 대댓글 모드 전환
- [ ] 입력창에 "@원작성자님에게 답글" 표시
- [ ] 취소 버튼으로 일반 모드 복귀
- [ ] 대댓글 생성 시 원 댓글 아래 표시

### AC-F6-5: 댓글 좋아요
- [ ] 댓글에 좋아요 버튼 표시
- [ ] 탭 시 좋아요 상태 토글
- [ ] 좋아요 수 즉시 UI 반영
