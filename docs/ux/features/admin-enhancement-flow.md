# F18: 관리자 기능 강화 - UX Flow

## 1. 개요

관리자의 오늘의 곡 등록을 상세화하고, 예약 게시와 댓글 관리 기능을 추가한다.

---

## 2. 사용자 여정

### 2.1 오늘의 곡 등록 (AE-01)

```
관리자 대시보드
  → "곡 관리" 메뉴 선택
  → 곡 등록 화면 (AE-01)
    → 제목 입력 (한국어/영어/일본어)
    → 아티스트 검색 (자동완성)
    → 스토리 입력 (에디터 추천 이유)
    → 가사 입력 (일본어 + 한국어)
    → YouTube URL 입력
    → 연관곡 추가 (검색 → 선택 → 이유 입력)
    → "저장" 버튼
  → 성공 → 곡 목록으로 이동
```

### 2.2 예약 게시 (AE-02)

```
관리자 대시보드
  → "예약 게시" 메뉴 선택
  → 예약 게시 화면 (AE-02)
    → 등록된 곡 목록 표시
    → 곡 선택
    → 날짜 선택 (DatePicker)
    → 시간 선택 (TimePicker)
    → 미리보기 확인
    → "예약 게시" 버튼 (곡 정보 미완성 시 비활성)
  → 성공 → 예약 목록에 추가됨
```

### 2.3 댓글 관리 (AE-03)

```
관리자 대시보드
  → "댓글 관리" 메뉴 선택
  → 댓글 관리 화면 (AE-03)
    → 댓글 목록 (페이지네이션)
      → 작성자, 내용 미리보기, 작성일 표시
    → 필터: 전체 / 신고된 댓글
    → 댓글 삭제
      → 확인 다이얼로그
      → 삭제 완료 → 목록 갱신
```
