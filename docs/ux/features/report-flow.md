# F11: Report (신고) 사용자 흐름

## 개요
부적절한 게시글, 댓글, 사용자를 신고하여 커뮤니티의 건전한 환경을 유지하는 기능

---

## Acceptance Criteria

### AC-F11-1: 게시글 신고
- **Given**: 로그인된 사용자
- **When**: 게시글의 더보기 메뉴에서 "신고하기" 선택
- **Then**: 신고 사유 선택 화면이 표시됨

### AC-F11-2: 댓글 신고
- **Given**: 로그인된 사용자
- **When**: 댓글의 더보기 메뉴에서 "신고하기" 선택
- **Then**: 신고 사유 선택 화면이 표시됨

### AC-F11-3: 사용자 신고
- **Given**: 로그인된 사용자
- **When**: 사용자 프로필의 더보기 메뉴에서 "신고하기" 선택
- **Then**: 신고 사유 선택 화면이 표시됨

### AC-F11-4: 신고 사유 선택
- **Given**: 신고 화면
- **When**: 신고 사유 선택 (스팸/광고, 욕설/비방, 불쾌한 내용, 저작권 침해, 기타)
- **Then**: 선택한 사유가 표시되고 제출 버튼 활성화

### AC-F11-5: 신고 상세 내용 입력
- **Given**: "기타" 사유 선택
- **When**: 상세 내용 입력
- **Then**: 입력 내용이 저장됨 (선택 사항)

### AC-F11-6: 신고 제출
- **Given**: 신고 사유가 선택된 상태
- **When**: 제출 버튼 클릭
- **Then**: 신고가 접수되고 확인 메시지 표시

### AC-F11-7: 중복 신고 방지
- **Given**: 이미 신고한 콘텐츠/사용자
- **When**: 다시 신고 시도
- **Then**: "이미 신고한 항목입니다" 메시지 표시

### AC-F11-8: 본인 콘텐츠 신고 방지
- **Given**: 로그인된 사용자
- **When**: 본인 게시글/댓글에서 신고 메뉴 확인
- **Then**: 신고하기 메뉴가 표시되지 않음

---

## 사용자 여정

### 진입점
1. 게시글 상세 화면 > 더보기 메뉴 (⋮) > 신고하기
2. 댓글 > 길게 누르기/더보기 메뉴 > 신고하기
3. 사용자 프로필 화면 > 더보기 메뉴 (⋮) > 신고하기

## 신고 대상 유형

| 대상 | 설명 | 진입점 |
|------|------|--------|
| POST | 피드 게시글 | 게시글 상세 > 더보기 메뉴 |
| COMMENT | 댓글/대댓글 | 댓글 > 더보기 메뉴 |
| MEMBER | 사용자 | 프로필 > 더보기 메뉴 |

---

## 신고 사유 정의

| 사유 | 코드 | 설명 |
|------|------|------|
| 스팸/광고 | SPAM | 상업적 광고, 스팸성 콘텐츠 |
| 욕설/비방 | ABUSE | 욕설, 비방, 혐오 발언 |
| 불쾌한 내용 | INAPPROPRIATE | 선정적, 폭력적, 불쾌한 콘텐츠 |
| 저작권 침해 | COPYRIGHT | 무단 복제, 저작권 침해 |
| 기타 | OTHER | 위 사유에 해당하지 않는 경우 |

---

## AC ↔ 화면 매핑

| AC | 화면 | 검증 방법 |
|----|------|----------|
| AC-F11-1 | 게시글 상세 → RP-01 | 게시글 더보기에서 신고 메뉴 표시 |
| AC-F11-2 | 댓글 → RP-01 | 댓글 더보기에서 신고 메뉴 표시 |
| AC-F11-3 | 프로필 → RP-01 | 프로필 더보기에서 신고 메뉴 표시 |
| AC-F11-4 | RP-01 | 사유 선택 시 체크 표시 및 제출 버튼 활성화 |
| AC-F11-5 | RP-01 | "기타" 선택 시 상세 입력 필드 표시 |
| AC-F11-6 | RP-01 → RP-02 | 제출 후 완료 화면 표시 |
| AC-F11-7 | RP-01 | 중복 신고 시 안내 다이얼로그 |
| AC-F11-8 | 각 더보기 메뉴 | 본인 콘텐츠에는 신고 메뉴 미표시 |

---

## 연관 기능
- **F5: Post** - 게시글 신고
- **F6: Comment** - 댓글 신고
- **F7: Follow** - 사용자 프로필에서 신고
- **F12: Management** - 관리자가 신고 내역 검토 및 처리
