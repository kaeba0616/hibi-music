# F17: My Page Enhancement (마이페이지 강화) 사용자 흐름

## 개요
마이페이지에 내가 쓴 댓글 목록과 푸시 알림 설정 기능을 추가하고, 문의 작성 시 일일 제한을 적용한다.

---

## Acceptance Criteria

### AC-MP-1: 내가 쓴 댓글 목록 조회
- **Given**: 로그인된 사용자
- **When**: 마이페이지에서 "내가 쓴 댓글" 메뉴 클릭
- **Then**: 본인이 작성한 댓글 목록이 표시됨 (댓글 내용, 곡 제목, 작성일)

### AC-MP-2: 내 댓글에서 해당 곡으로 이동
- **Given**: 내가 쓴 댓글 목록 화면
- **When**: 특정 댓글 항목 탭
- **Then**: 해당 곡의 상세 페이지로 이동 (댓글 섹션)

### AC-MP-3: 푸시 알림 on/off 토글
- **Given**: 로그인된 사용자
- **When**: 설정 화면에서 푸시 알림 토글 변경
- **Then**: 알림 수신 여부가 변경됨 (서버에 저장)

### AC-MP-4: 문의 작성 제한 - 최소 글자 수
- **Given**: 문의 작성 화면
- **When**: 내용 10자 미만으로 제출 시도
- **Then**: "내용은 최소 10자 이상 입력해주세요" 에러 메시지 표시

### AC-MP-5: 문의 작성 제한 - 일일 3개
- **Given**: 오늘 이미 3개의 문의를 작성한 사용자
- **When**: 추가 문의 작성 시도
- **Then**: "오늘의 문의 작성 한도(3개)를 초과했습니다" 에러 메시지 표시

---

## 사용자 여정

### Journey 1: 내가 쓴 댓글 확인
```
마이페이지 → "내가 쓴 댓글" 탭 → 댓글 목록 표시
→ 댓글 탭 → 해당 곡 상세 페이지 이동
```

### Journey 2: 푸시 알림 설정
```
마이페이지 → 설정 (톱니바퀴) → 푸시 알림 토글 → on/off 변경
```

### Journey 3: 문의 작성 (제한 적용)
```
문의하기 → 유형/제목/내용 입력 → 제출
→ [내용 10자 미만] 에러 표시
→ [오늘 3개 초과] 에러 표시
→ [정상] 문의 접수 완료
```
