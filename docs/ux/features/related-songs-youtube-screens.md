## 1. DS-02 확장: 노래 상세 화면

### 기본 정보
- Route: `/songs/:songId` (기존 유지)
- 변경 사항: 기존 화면에 유튜브 영상 + 연관곡 섹션 추가
- 기존 파일: `hibi_front/lib/features/daily-song/views/song_detail_view.dart`

### 유튜브 섹션 UI 요소 (AC-F2-5)

| 요소 | 타입 | 설명 |
|------|------|------|
| 섹션 헤더 | Text | "🎬 뮤직비디오" |
| 유튜브 플레이어 | YouTubePlayer | 16:9 비율, 인라인 재생 |
| 재생 버튼 | Overlay | 썸네일 위에 재생 아이콘 |

#### 유튜브 상태별 UI
| 상태 | UI |
|------|-----|
| 로딩 | 16:9 Shimmer 영역 |
| 성공 | 유튜브 영상 썸네일 + 재생 버튼 |
| URL 없음 | 섹션 자체 숨김 |
| 로딩 실패 | "영상을 불러올 수 없습니다" + "YouTube에서 보기" 버튼 |

### 연관곡 섹션 UI 요소 (AC-F2-6)

| 요소 | 타입 | 설명 |
|------|------|------|
| 섹션 헤더 | Text | "🎵 연관곡" |
| 연관곡 리스트 | ListView | 최대 5개 표시 |
| RelatedSongTile | ListTile | 앨범 이미지 + 제목 + 아티스트 + 선정 이유 |

#### RelatedSongTile 구성
```
┌──────────────────────────────────┐
│ [앨범]  곡 제목 (한글)            │
│  48x48  아티스트명            >  │
│         "선정 이유 텍스트"        │  ← 작은 폰트, 회색
└──────────────────────────────────┘
```

| 요소 | 타입 | 설명 |
|------|------|------|
| 앨범 이미지 | Image (48x48) | 원형 또는 사각형 |
| 곡 제목 | Text | 1줄, overflow ellipsis |
| 아티스트명 | Text | 작은 폰트, 회색 |
| 선정 이유 | Text | 이탤릭, 작은 폰트 |
| 화살표 | Icon | chevron_right |

#### 연관곡 상태별 UI
| 상태 | UI |
|------|-----|
| 로딩 | Shimmer 3줄 |
| 성공 | RelatedSongTile 리스트 |
| 연관곡 없음 | 섹션 자체 숨김 |

#### 인터랙션
| 액션 | 트리거 | 결과 |
|------|--------|------|
| 연관곡 탭 | RelatedSongTile 탭 | 해당 곡의 DS-02 상세 화면으로 이동 |
| 유튜브 재생 | 재생 버튼 탭 | 인라인 영상 재생 |
| 유튜브 전체화면 | 전체화면 버튼 | 가로 모드 전체화면 재생 |

## 2. RS-01: 좋아요 곡 모아보기 (신규)

### 기본 정보
- Route: `/liked-songs`
- 타입: 목록
- 진입점: 마이페이지 또는 네비게이션
- 인증 필요: Yes

### UI 요소 상세

| 요소 | 타입 | 설명 |
|------|------|------|
| AppBar | AppBar | 뒤로가기 + "좋아요 곡" 타이틀 |
| 곡 수 | Text | "총 N곡" |
| LikedSongTile | ListTile | 앨범 이미지 + 제목 + 아티스트 + 날짜 + 좋아요 아이콘 |

#### LikedSongTile 구성
```
┌──────────────────────────────────┐
│ [앨범]  곡 제목                   │
│  48x48  아티스트명            >  │
│         2026.02.03    ♡         │  ← 추천 날짜 + 좋아요(빨간)

```

### 인터랙션
| 액션 | 트리거 | 결과 |
|------|--------|------|
| 곡 탭 | LikedSongTile 탭 | 해당 곡의 DS-02 상세 화면으로 이동 |
| 좋아요 해제 | 좋아요 아이콘 탭 | 확인 다이얼로그 후 목록에서 제거 |
| 뒤로가기 | 뒤로 버튼 | 이전 화면 복귀 |
| 스크롤 | 아래로 스크롤 | 페이지네이션 (무한 스크롤) |
| "오늘의 곡 보러가기" | Empty 상태 CTA | 메인 화면(DS-01)으로 이동 |

## AC 검증 체크리스트

### AC-F2-5: 유튜브 영상
- [ ] DS-02에 유튜브 영상 플레이어가 표시됨
- [ ] 재생 버튼 탭 시 인라인 재생
- [ ] 유튜브 URL 없는 곡은 섹션 숨김
- [ ] 로딩 실패 시 대체 UI 표시

### AC-F2-6: 연관곡 표시
- [ ] DS-02 하단에 연관곡 리스트 표시
- [ ] 각 연관곡에 제목, 아티스트, 선정 이유 표시
- [ ] 연관곡 탭 시 해당 곡 상세 화면으로 이동
- [ ] 연관곡 없으면 섹션 숨김

### AC-F2-7: 좋아요 곡 모아보기
- [ ] RS-01에 좋아요 곡 목록 표시 (제목, 아티스트)
- [ ] 곡 탭 시 상세 화면 이동
- [ ] 좋아요 곡 없으면 Empty 상태 표시
- [ ] 무한 스크롤 페이지네이션
