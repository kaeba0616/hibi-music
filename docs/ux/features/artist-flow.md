# Artist (F3) - UX Flow

## 1. 개요

### 1.1 기능 설명
아티스트 목록을 조회하고, 상세 정보를 확인하며, 팔로우할 수 있는 기능

### 1.2 관련 AC
- AC-F3-1: 아티스트 목록 조회
- AC-F3-2: 아티스트 상세 조회
- AC-F3-3: 아티스트 팔로우
- AC-F3-4: 팔로우 중인 아티스트 필터

---

## 2. 사용자 여정 (User Journey)

### 2.1 Entry Points
| 진입점 | 조건 | 목적지 |
|--------|------|--------|
| 하단 탭 "아티스트" 클릭 | 로그인 상태 | 아티스트 목록 화면 |
| Daily Song 아티스트명 클릭 | 로그인 상태 | 아티스트 상세 화면 |
| 검색 결과에서 아티스트 클릭 | 로그인 상태 | 아티스트 상세 화면 |

### 2.2 Exit Points
| 종료점 | 조건 | 다음 목적지 |
|--------|------|------------|
| 노래 클릭 | 곡 목록에서 | 노래 상세 화면 (F2) |
| 뒤로가기 | 상세 화면에서 | 이전 화면 |
| 다른 탭 선택 | 어디서나 | 해당 탭 화면 |

## 4. 화면 목록

| 화면 ID | 화면명 | Route | 설명 |
|---------|--------|-------|------|
| AR-01 | 아티스트 목록 | `/artists` (탭) | 아티스트 그리드/리스트 표시 |
| AR-02 | 아티스트 상세 | `/artists/:artistId` | 아티스트 상세 정보 표시 |

## 5. 주요 인터랙션

### 5.1 팔로우 토글
```
[팔로우 버튼 탭]
    │
    ├─ 현재 팔로우 상태 확인
    │
    ├─ 낙관적 UI 업데이트 (즉시 반영)
    │
    ├─ API 호출
    │
    └─ 실패 시 롤백
```

### 5.2 아티스트 목록 필터
```
[필터 탭 선택]
    │
    ├─ "전체" 선택 → 모든 아티스트 표시
    │
    └─ "팔로우 중" 선택 → 팔로우한 아티스트만 표시
```

### 5.3 검색
```
[검색어 입력]
    │
    ├─ 디바운스 적용 (300ms)
    │
    ├─ 검색 API 호출
    │
    └─ 결과 표시 (또는 "검색 결과 없음")
```

## 6. 에러 처리

| 에러 상황 | 사용자에게 보여줄 내용 | 액션 |
|-----------|----------------------|------|
| 네트워크 에러 | "연결을 확인해주세요" | 재시도 버튼 |
| 서버 에러 | "잠시 후 다시 시도해주세요" | 재시도 버튼 |
| 팔로우 실패 | 토스트 "팔로우 처리 실패" | 자동 롤백 |
| 아티스트 없음 | "등록된 아티스트가 없습니다" | - |
