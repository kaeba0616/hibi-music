# Daily Song (F2) - UX Flow

## 1. 개요

### 1.1 기능 설명
매일 추천되는 JPOP 한 곡을 확인하고 좋아요를 표시하는 핵심 기능

### 1.2 관련 AC
- AC-F2-1: 오늘의 노래 표시
- AC-F2-2: 노래 좋아요
- AC-F2-3: 노래 상세 정보
- AC-F2-4: 추천곡 없음

---

## 2. 사용자 여정 (User Journey)

### 2.1 Entry Points
| 진입점 | 조건 | 목적지 |
|--------|------|--------|
| 앱 시작 | 로그인 상태 | 홈 화면 |
| 하단 탭 "홈" 클릭 | 로그인 상태 | 홈 화면 |
| 푸시 알림 클릭 | 로그인 상태 | 노래 상세 화면 (향후) |

### 2.2 Exit Points
| 종료점 | 조건 | 다음 목적지 |
|--------|------|------------|
| 아티스트 이름 클릭 | 아티스트 정보 존재 | 아티스트 상세 (F3) |
| 뒤로가기 | 상세 화면에서 | 홈 화면 |

## 4. 화면 목록

| 화면 ID | 화면명 | Route | 설명 |
|---------|--------|-------|------|
| DS-01 | 홈 화면 | `/home` | 오늘의 노래 카드 표시 |
| DS-02 | 노래 상세 | `/songs/:songId` | 노래 상세 정보 표시 |

## 5. 주요 인터랙션

### 5.1 좋아요 토글
```
[좋아요 버튼 탭]
    │
    ├─ 현재 좋아요 상태 확인
    │
    ├─ 낙관적 UI 업데이트 (즉시 반영)
    │
    ├─ API 호출
    │
    └─ 실패 시 롤백
```

### 5.2 카드 → 상세 화면 전환
```
[카드 탭]
    │
    ├─ 카드 확대 애니메이션 (Hero)
    │
    ├─ 상세 화면으로 전환
    │
    └─ 앨범 이미지가 확대되며 연결
```

---

## 6. 에러 처리

| 에러 상황 | 사용자에게 보여줄 내용 | 액션 |
|-----------|----------------------|------|
| 네트워크 에러 | "연결을 확인해주세요" | 재시도 버튼 |
| 서버 에러 | "잠시 후 다시 시도해주세요" | 재시도 버튼 |
| 좋아요 실패 | 토스트 "좋아요 처리 실패" | 자동 롤백 |

## 7. AC 매핑 요약

| AC ID | 검증 화면 | 검증 포인트 |
|-------|----------|------------|
| AC-F2-1 | DS-01 홈 화면 | 곡명, 아티스트, 앨범 이미지 표시 |
| AC-F2-2 | DS-01, DS-02 | 좋아요 버튼 토글 동작 |
| AC-F2-3 | DS-02 상세 화면 | 가사, 발매일, 앨범 정보 표시 |
| AC-F2-4 | DS-01 홈 화면 | Empty 상태 메시지 표시 |
