# Search (F8) - UX Flow

## 1. 개요

### 1.1 기능 설명
노래, 아티스트, 게시글, 사용자를 통합 검색하여 원하는 콘텐츠를 빠르게 찾을 수 있는 기능

### 1.2 관련 AC
- AC-F8-1: 통합 검색
- AC-F8-2: 카테고리별 필터
- AC-F8-3: 최근 검색어
- AC-F8-4: 검색 결과 없음
- AC-F8-5: 검색 결과 상세 이동

---

## 2. Acceptance Criteria

### AC-F8-1: 통합 검색
- Given: 로그인된 사용자
- When: 검색창에 키워드 입력 후 검색
- Then: 노래, 아티스트, 게시글, 사용자 결과가 카테고리별로 표시됨

### AC-F8-2: 카테고리별 필터
- Given: 검색 결과가 표시된 상태
- When: 특정 카테고리 탭 선택 (노래/아티스트/게시글/사용자)
- Then: 해당 카테고리의 결과만 표시됨

### AC-F8-3: 최근 검색어
- Given: 검색 화면 진입
- When: 검색창에 포커스
- Then: 최근 검색어 목록이 표시됨 (최대 10개)

### AC-F8-4: 검색 결과 없음
- Given: 검색 실행
- When: 검색 결과가 없는 경우
- Then: "검색 결과가 없습니다" 메시지와 다른 키워드 제안

### AC-F8-5: 검색 결과 상세 이동
- Given: 검색 결과가 표시된 상태
- When: 결과 항목 클릭
- Then: 해당 항목의 상세 화면으로 이동

---

## 3. 사용자 여정 (User Journey)

### 3.1 Entry Points
| 진입점 | 조건 | 목적지 |
|--------|------|--------|
| 하단 탭 "검색" 클릭 | 로그인 상태 | 검색 화면 |
| 앱바 검색 아이콘 클릭 | 어디서나 | 검색 화면 |

### 3.2 Exit Points
| 종료점 | 조건 | 다음 목적지 |
|--------|------|------------|
| 노래 결과 클릭 | 검색 결과에서 | 노래 상세 화면 (F2) |
| 아티스트 결과 클릭 | 검색 결과에서 | 아티스트 상세 화면 (F3) |
| 게시글 결과 클릭 | 검색 결과에서 | 게시글 상세 화면 (F5) |
| 사용자 결과 클릭 | 검색 결과에서 | 사용자 프로필 화면 (F7) |
| 뒤로가기 | 어디서나 | 이전 화면 |
| 다른 탭 선택 | 어디서나 | 해당 탭 화면 |

## 6. 주요 인터랙션

### 6.1 실시간 검색 (디바운스)
```
[검색어 입력]
    │
    ├─ 디바운스 적용 (300ms)
    │
    ├─ 2글자 이상일 때만 검색
    │
    ├─ 통합 검색 API 호출
    │
    └─ 결과 표시 (카테고리별 상위 N개)
```

### 6.2 카테고리 탭 전환
```
[카테고리 탭 선택]
    │
    ├─ "전체" → 모든 카테고리 결과 표시 (미리보기)
    │
    ├─ "노래" → 노래 검색 결과만 표시
    │
    ├─ "아티스트" → 아티스트 검색 결과만 표시
    │
    ├─ "게시글" → 게시글 검색 결과만 표시
    │
    └─ "사용자" → 사용자 검색 결과만 표시
```

### 6.3 최근 검색어 관리
```
[최근 검색어]
    │
    ├─ 검색 시 자동 저장 (최대 10개)
    │
    ├─ 클릭 → 해당 키워드로 검색
    │
    ├─ X 버튼 → 개별 삭제
    │
    └─ "전체 삭제" → 모든 검색어 삭제
```

### 6.4 검색 결과 클릭
```
[검색 결과 항목 클릭]
    │
    ├─ 노래 → /songs/:songId (노래 상세)
    │
    ├─ 아티스트 → /artists/:artistId (아티스트 상세)
    │
    ├─ 게시글 → /posts/:postId (게시글 상세)
    │
    └─ 사용자 → /users/:userId (사용자 프로필)
```

## 7. 에러 처리

| 에러 상황 | 사용자에게 보여줄 내용 | 액션 |
|-----------|----------------------|------|
| 네트워크 에러 | "연결을 확인해주세요" | 재시도 버튼 |
| 서버 에러 | "잠시 후 다시 시도해주세요" | 재시도 버튼 |
| 검색 결과 없음 | "검색 결과가 없습니다" | 다른 키워드 제안 |
| 검색어 너무 짧음 | "2글자 이상 입력해주세요" | - |
