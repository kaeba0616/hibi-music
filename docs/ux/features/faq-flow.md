# F9: FAQ 사용자 흐름

## 개요
자주 묻는 질문(FAQ)을 통해 사용자가 서비스 이용 중 궁금한 점을 쉽게 해결할 수 있는 기능

---

## Acceptance Criteria

### AC-F9-1: FAQ 목록 조회
- **Given**: 로그인/비로그인 사용자
- **When**: 설정 > FAQ 메뉴 진입
- **Then**: FAQ 목록이 카테고리별로 그룹화되어 표시됨

### AC-F9-2: FAQ 카테고리 필터
- **Given**: FAQ 목록 화면
- **When**: 특정 카테고리 탭 선택
- **Then**: 해당 카테고리의 FAQ만 표시됨

### AC-F9-3: FAQ 상세 보기
- **Given**: FAQ 목록이 표시된 상태
- **When**: FAQ 항목 클릭
- **Then**: 질문과 답변이 확장되어 표시됨 (아코디언 방식)

### AC-F9-4: FAQ 검색
- **Given**: FAQ 화면
- **When**: 검색창에 키워드 입력
- **Then**: 질문/답변에 키워드가 포함된 FAQ가 필터링되어 표시됨

### AC-F9-5: FAQ 없음 상태
- **Given**: FAQ 화면
- **When**: 검색 결과가 없거나 등록된 FAQ가 없는 경우
- **Then**: "검색 결과가 없습니다" 또는 "등록된 FAQ가 없습니다" 메시지 표시

---

## 사용자 여정

### 진입점
1. 설정 화면 > FAQ 메뉴
2. 앱 하단 > 고객센터/도움말 링크

## 카테고리 정의

| 카테고리 | 설명 | 예시 질문 |
|---------|------|----------|
| 계정 | 가입, 로그인, 탈퇴 관련 | "비밀번호를 잊어버렸어요" |
| 서비스 이용 | 기능 사용법 | "오늘의 노래는 어떻게 확인하나요?" |
| 커뮤니티 | 게시글, 댓글, 팔로우 | "게시글은 어떻게 작성하나요?" |
| 기타 | 그 외 문의 | "앱 버전은 어디서 확인하나요?" |

---

## AC ↔ 화면 매핑

| AC | 화면 | 검증 방법 |
|----|------|----------|
| AC-F9-1 | FA-01 | FAQ 목록이 카테고리별로 그룹화되어 표시 |
| AC-F9-2 | FA-01 | 카테고리 탭 클릭 시 해당 카테고리만 표시 |
| AC-F9-3 | FA-01 | FAQ 항목 클릭 시 답변 확장 |
| AC-F9-4 | FA-01 | 검색창 입력 시 필터링 |
| AC-F9-5 | FA-01 | 결과 없을 때 Empty View 표시 |

---

## 연관 기능
- **F10: Question (문의하기)** - FAQ에서 해결되지 않은 경우 문의하기로 이동
- **F1: Auth** - 로그인 관련 FAQ에서 로그인 화면으로 이동 가능
