# F12: Management (관리자 기능) 사용자 흐름

## 개요
관리자가 서비스를 운영하고 콘텐츠를 관리하는 기능

---

## Acceptance Criteria

### AC-F12-1: 관리자 대시보드 접근
- **Given**: ADMIN 권한을 가진 사용자
- **When**: 관리자 메뉴에 접근
- **Then**: 관리자 대시보드가 표시됨 (통계 요약)

### AC-F12-2: 신고 목록 조회
- **Given**: 관리자 대시보드
- **When**: 신고 관리 메뉴 선택
- **Then**: 대기중인 신고 목록이 표시됨 (최신순)

### AC-F12-3: 신고 상세 조회
- **Given**: 신고 목록 화면
- **When**: 특정 신고 항목 클릭
- **Then**: 신고 상세 정보와 신고 대상 콘텐츠가 표시됨

### AC-F12-4: 신고 처리 (조치)
- **Given**: 신고 상세 화면
- **When**: 조치 버튼 클릭 (삭제, 경고, 기각)
- **Then**: 신고 상태가 변경되고 필요시 콘텐츠가 삭제됨

### AC-F12-5: 문의 목록 조회
- **Given**: 관리자 대시보드
- **When**: 문의 관리 메뉴 선택
- **Then**: 미답변 문의 목록이 표시됨 (접수순)

### AC-F12-6: 문의 답변 작성
- **Given**: 문의 상세 화면
- **When**: 답변 작성 후 제출
- **Then**: 답변이 저장되고 문의 상태가 '답변완료'로 변경됨

### AC-F12-7: FAQ 목록 관리
- **Given**: 관리자 대시보드
- **When**: FAQ 관리 메뉴 선택
- **Then**: FAQ 목록이 표시됨 (카테고리별)

### AC-F12-8: FAQ 추가/수정/삭제
- **Given**: FAQ 관리 화면
- **When**: 추가/수정/삭제 버튼 클릭
- **Then**: FAQ 항목이 추가/수정/삭제됨

### AC-F12-9: 사용자 목록 조회
- **Given**: 관리자 대시보드
- **When**: 사용자 관리 메뉴 선택
- **Then**: 사용자 목록이 표시됨 (검색/필터 가능)

### AC-F12-10: 사용자 제재 (정지/탈퇴)
- **Given**: 사용자 상세 화면
- **When**: 제재 버튼 클릭
- **Then**: 사용자 계정이 정지되거나 강제 탈퇴됨

### AC-F12-11: 일반 사용자 접근 차단
- **Given**: USER 권한을 가진 사용자
- **When**: 관리자 URL 직접 접근 시도
- **Then**: 403 Forbidden 또는 메인 화면으로 리다이렉트

## 관리 대상별 상태

### 신고 상태 흐름
```
PENDING (대기중) → REVIEWED (검토됨) → RESOLVED (처리완료)
                                    ↘ DISMISSED (기각)
```

### 문의 상태 흐름
```
RECEIVED (접수됨) → PROCESSING (처리중) → ANSWERED (답변완료)
```

### 사용자 상태
```
ACTIVE (활성) → SUSPENDED (정지) → BANNED (탈퇴)
```

## 관리 메뉴 구조

| 메뉴 | 설명 | 화면 ID |
|------|------|---------|
| 대시보드 | 통계 요약 (오늘 신고, 미답변 문의 등) | MG-01 |
| 신고 관리 | 신고 목록/상세/처리 | MG-02, MG-03 |
| 문의 관리 | 문의 목록/상세/답변 | MG-04, MG-05 |
| FAQ 관리 | FAQ 목록/추가/수정/삭제 | MG-06, MG-07 |
| 사용자 관리 | 사용자 목록/상세/제재 | MG-08, MG-09 |

## 연관 기능
- **F9: FAQ** - FAQ 조회 (사용자용) ↔ FAQ 관리 (관리자용)
- **F10: Question** - 문의 작성 (사용자용) ↔ 문의 답변 (관리자용)
- **F11: Report** - 신고 생성 (사용자용) ↔ 신고 처리 (관리자용)
- **F1: Auth** - 사용자 프로필 ↔ 사용자 제재
