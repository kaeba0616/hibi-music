# F15: 연관곡 & 유튜브 임베드 - UX Flow

## 1. 개요
- Feature ID: F15
- Feature 이름: 연관곡 & 유튜브 임베드
- 설명: 오늘의 곡 상세 화면에 유튜브 영상을 임베드하고, 연관곡 목록을 추가하며, 좋아요 곡 모아보기 페이지를 구현한다.
- Phase: 4 (Enhancement & Notion PRD Sync)
- 기존 화면 확장: DS-02 (노래 상세 화면)

---

## 2. Acceptance Criteria

### AC-F2-5: 유튜브 영상
- Given: 오늘의 노래 상세 화면
- When: 화면 진입
- Then: 해당 노래의 유튜브 영상이 임베드되어 표시됨

### AC-F2-6: 연관곡 표시
- Given: 오늘의 노래 상세 화면 하단
- When: 연관곡 영역 확인
- Then: 연관곡 목록이 표시됨 (제목, 아티스트, 선정 이유), 클릭 시 해당 곡 소개 페이지로 이동

### AC-F2-7: 좋아요 곡 모아보기
- Given: 로그인된 사용자
- When: 좋아요 곡 목록 화면 진입
- Then: 좋아요 누른 오늘의 곡 목록이 표시됨 (제목, 아티스트)

---

## 3. 사용자 여정

### 3.1 유튜브 영상 시청 플로우
```
DS-02 노래 상세 화면 진입
  └─ 스크롤하여 유튜브 영역 도달
      ├─ 유튜브 영상 썸네일 표시
      ├─ 재생 버튼 탭 → 인라인 재생 시작
      └─ 유튜브 영상 없음 → 해당 섹션 비표시
```

### 3.2 연관곡 탐색 플로우
```
DS-02 노래 상세 화면 진입
  └─ 하단 스크롤하여 연관곡 섹션 도달
      ├─ 연관곡 목록 표시 (제목, 아티스트, 선정 이유)
      ├─ 연관곡 항목 탭 → 해당 곡 상세 화면(DS-02)으로 이동
      └─ 연관곡 없음 → 해당 섹션 비표시
```

### 3.3 좋아요 곡 모아보기 플로우
```
하단 네비게이션 또는 마이페이지
  └─ 좋아요 곡 목록 화면 (RS-01) 진입
      ├─ 좋아요 누른 곡 목록 표시 (제목, 아티스트, 날짜)
      ├─ 곡 항목 탭 → 해당 곡 상세 화면(DS-02)으로 이동
      └─ 좋아요 곡 없음 → Empty 상태 표시
```

### 3.4 대체 플로우

| 상황 | 처리 |
|------|------|
| 유튜브 URL 없음 | 유튜브 섹션 자체를 숨김 |
| 연관곡 없음 | 연관곡 섹션 자체를 숨김 |
| 좋아요 곡 없음 | Empty 상태 ("아직 좋아요한 곡이 없어요") |
| 유튜브 로딩 실패 | "영상을 불러올 수 없습니다" + 유튜브 앱 열기 버튼 |
