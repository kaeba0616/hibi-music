# Follow (F7) - User Flow

## 1. Feature 개요

### 1.1 목표
사용자 간 팔로우 기능을 통해 관심 있는 사용자의 게시글을 모아볼 수 있는 소셜 기능

### 1.2 AC 목록
| AC ID | 설명 |
|-------|------|
| AC-F7-1 | 사용자 프로필 조회 |
| AC-F7-2 | 사용자 팔로우 |
| AC-F7-3 | 사용자 언팔로우 |
| AC-F7-4 | 팔로워 목록 조회 |
| AC-F7-5 | 팔로잉 목록 조회 |
| AC-F7-6 | 팔로우 피드 |
| AC-F7-7 | 사용자 게시글 목록 |

---

## 2. 사용자 여정

### 2.1 진입점
- **게시글 작성자 프로필 탭** (피드, 게시글 상세에서)
- **댓글 작성자 프로필 탭** (댓글 섹션에서)
- **팔로워/팔로잉 목록에서 사용자 탭**
- **검색 결과에서 사용자 탭** (F8에서)

### 2.2 주요 시나리오

#### 시나리오 1: 사용자 프로필 조회 (AC-F7-1, AC-F7-7)
```
[피드/게시글 상세]
       ↓
[작성자 프로필 탭]
       ↓
[사용자 프로필 화면]
       ↓
[프로필 정보 확인]
       ↓
[사용자 게시글 스크롤]
```

#### 시나리오 2: 사용자 팔로우 (AC-F7-2)
```
[사용자 프로필 화면]
       ↓
[팔로우 버튼 탭]
       ↓
[팔로우 완료]
       ↓
[버튼이 "팔로잉"으로 변경]
       ↓
[팔로워 수 +1]
```

#### 시나리오 3: 사용자 언팔로우 (AC-F7-3)
```
[사용자 프로필 화면]
       ↓
["팔로잉" 버튼 탭]
       ↓
[언팔로우 확인 다이얼로그]
       ↓
[확인 탭]
       ↓
[언팔로우 완료]
       ↓
[버튼이 "팔로우"로 변경]
```

#### 시나리오 4: 팔로워/팔로잉 목록 조회 (AC-F7-4, AC-F7-5)
```
[사용자 프로필 화면]
       ↓
[팔로워/팔로잉 수 탭]
       ↓
[목록 화면]
       ↓
[사용자 목록 스크롤]
       ↓
[사용자 탭 → 프로필 이동]
  또는
[팔로우/언팔로우 버튼 탭]
```

#### 시나리오 5: 팔로우 피드 (AC-F7-6)
```
[피드 화면 (PO-01)]
       ↓
[상단 "팔로잉" 탭 선택]
       ↓
[팔로우한 사용자 게시글만 표시]
       ↓
[팔로우 없으면 Empty View]
```

## 4. 상태 전이

### 4.1 팔로우 상태
```
[팔로우 안함] ←→ [팔로잉]
      ↓              ↓
  [팔로우 버튼]  [팔로잉 버튼 + 확인]
```

### 4.2 피드 필터 상태
```
[전체 피드] ←→ [팔로잉 피드]
```

## 5. AC → 화면 매핑

| AC ID | 화면 | 검증 포인트 |
|-------|------|-------------|
| AC-F7-1 | FO-01 (사용자 프로필) | 프로필 정보가 모두 표시됨 |
| AC-F7-2 | FO-01 (팔로우 버튼) | 버튼 탭 시 팔로잉 상태로 변경 |
| AC-F7-3 | FO-01 (팔로잉 버튼) | 확인 다이얼로그 후 팔로우 해제 |
| AC-F7-4 | FO-02 (팔로워 탭) | 팔로워 목록 표시 |
| AC-F7-5 | FO-02 (팔로잉 탭) | 팔로잉 목록 표시 |
| AC-F7-6 | PO-01 (팔로잉 필터) | 팔로우한 사용자 게시글만 표시 |
| AC-F7-7 | FO-01 (게시글 목록) | 해당 사용자의 게시글 표시 |

## 6. 에러 처리

### 6.1 프로필 로딩 실패
- 네트워크 오류 시 "프로필을 불러올 수 없습니다" 표시
- 재시도 버튼 제공

### 6.2 팔로우 실패
- 네트워크 오류 시 "팔로우에 실패했습니다" 토스트
- 버튼 상태 롤백

### 6.3 팔로잉 피드 빈 상태
- "팔로우한 사용자가 없습니다"
- "사용자를 팔로우하고 게시글을 모아보세요!"
- 사용자 검색 버튼 (F8 연동 예정)

---

## 7. 특이사항

### 7.1 본인 프로필 vs 타인 프로필
- 본인 프로필: 팔로우 버튼 대신 "프로필 수정" 버튼
- 타인 프로필: 팔로우/팔로잉 버튼

### 7.2 언팔로우 확인
- 실수 방지를 위해 언팔로우 시 확인 다이얼로그 표시
- "정말 언팔로우하시겠습니까?"

### 7.3 팔로워/팔로잉 목록에서 팔로우
- 목록 내에서 바로 팔로우/언팔로우 가능
- 상태 변경 시 즉시 UI 반영

### 7.4 낙관적 업데이트
- 팔로우/언팔로우 시 즉시 UI 반영
- 실패 시 롤백 및 에러 토스트
