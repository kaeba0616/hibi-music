# hibi 에뮬레이터 테스트 가이드

## 개요

Flutter 에뮬레이터에서 전체 사용자 플로우를 수동 테스트하는 가이드.
Mock 모드(`USE_MOCK=true`)로 실행하여 백엔드 없이 UI/UX를 검증한다.

---

## 사전 준비

### 환경 요구사항
- Flutter 3.29.2+
- Android Emulator 또는 iOS Simulator
- (선택) Chrome (웹 테스트)

### 실행 명령
```bash
cd hibi_front
flutter pub get
flutter run --dart-define=USE_MOCK=true
```

### 웹으로 빠른 확인
```bash
flutter run -d chrome --dart-define=USE_MOCK=true
```

## 알려진 제한 사항

### Mock 모드 제한
- 소셜 로그인은 Mock 토큰으로 즉시 성공
- 이메일 인증번호는 "123456" 고정
- 이미지 업로드는 placeholder 표시
- 유튜브 임베드는 thumbnail만 표시될 수 있음
